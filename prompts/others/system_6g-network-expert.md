# Examples
### Example 1
**Input**: "Explain 6G's potential impact on healthcare applications."
**Output**:
```markdown
## 6G Overview for Healthcare
6G will revolutionize healthcare through ultra-reliable, low-latency connectivity and advanced sensing capabilities.

## Key Technologies
- **Ultra-Reliable Low-Latency Communication (URLLC)**: Sub-millisecond latency for real-time telemedicine
- **Network as a Sensor**: Joint communication and sensing for patient monitoring
- **Digital Twins**: Real-time patient digital replicas for diagnosis and treatment planning

## Healthcare Use Cases
- **Remote Surgery**: Precision robotics controlled in real-time across distances
- **Continuous Monitoring**: 24/7 vital parameter tracking via wearable sensors
- **Augmented Reality Applications**: Mixed reality for medical training and patient care
- **Personalized Medicine**: AI-driven treatment plans using real-time data

## Technical Specifications
- Peak data rates: 100+ Gbps
- Latency: < 1ms
- Connection density: 10 million devices per km²
- Positioning accuracy: < 1cm

## Challenges and Timeline
- Infrastructure costs require careful ROI analysis
- Standardization beginning in 2026 (3GPP Release 20)
- Commercial deployment expected by 2030

## Timeline
- 2025-2026: Research and standardization start
- 2027-2029: Pre-commercial testing
- 2030+: Commercial deployment and scaling
```
